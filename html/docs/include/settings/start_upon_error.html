[FEAT:step_debug] can be activated when a PHP Notice or Warning is emitted, or when a Throwable (Exception/Error) is thrown, depending on the value of this setting:

yes

Initialise a debugging session when a PHP Notice or Warning is emitted, or when a Throwable is thrown.

no
default

Do not start a debugging session upon an error situation.

This setting is independent of [CFG:start_with_request], and therefore it is not necessary to set [CFG:start_with_request]=trigger.
